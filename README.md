## 📌 스터디 규칙
1. 진행 방식

    - 주마다 n문제씩 선정해서 풀이 진행
    - 스터디 진행 시 각자 1문제씩 풀이 설명
    - 개념 + 알고리즘 + 풀이 방식 자세하게 설명하기

2. 추후 추가 예정


## 🔍 참여 방법
1. 이 저장소를 `fork` 한다. 
2. 매주 새 폴더를 생성한다. (ex. `Week01`, `Week02` ...)
3. 디렉터리에 문제 폴더를 또 생성한다. 
4. 해당 문제 폴더에 자신의 소스코드 파일을 업로드 한다. 
5. 원본 저장소로 `Pull Request`를 한다. 
6. [참고링크](https://waytocse.tistory.com/59)


### ✅ 문제 폴더 이름 규칙
#### 주차/{문제번호}_{문제명}
- 문제명에 띄어쓰기 있을 시 '_'로 연결해주세요!
- ex. `Week01/1945_간단한_소인수분해`


### ✅ 소스코드 파일 이름 규칙
#### 문제번호_이름.py
- ex. `1945_송지영.py`
